# PracticaCSSFinal

## Descripción del proyecto

Este proyecto fue desarrollado como parte de la asignatura de **Fundamentos de Sistemas Web**, con el propósito de aplicar conocimientos relacionados con **HTML5, CSS3, estructura semántica, formularios, multimedia, navegación entre páginas y diseño web responsivo**.

El sitio web fue construido utilizando múltiples páginas conectadas mediante navegación interna y rutas relativas, incorporando diferentes componentes como formularios, tablas, imágenes, contenido multimedia y estilos adaptables mediante **media queries**.

Como extensión del proyecto original, se desarrolló una nueva página llamada **portafolio.html**, diseñada para presentar habilidades técnicas, proyectos académicos, progreso de aprendizaje y metas profesionales, manteniendo coherencia visual con el resto del sitio.

---

## Objetivo del proyecto

Aplicar los conceptos aprendidos en la Unidad 2 relacionados con:

- HTML semántico
- Sintaxis CSS
- Selectores
- Diseño visual
- Modelo de caja
- Flexbox
- CSS Grid
- Diseño responsivo
- Media Queries

---

## Páginas incluidas

El proyecto contiene las siguientes páginas:

### index.html
Página principal del sitio web donde se presenta información personal, intereses, multimedia, formularios, tablas y navegación interna.

### buscar.html
Página destinada a practicar formularios utilizando el método **GET**, mostrando cómo los datos son enviados mediante la URL.

### contacto.html
Página orientada a formularios avanzados con validaciones utilizando atributos HTML como:

- required
- minlength
- maxlength
- pattern

### portafolio.html
Nueva página desarrollada como extensión del proyecto principal, incluyendo:

- Tarjetas informativas
- Tabla de habilidades
- Imágenes
- Elementos semánticos
- Diseño responsivo
- Metas profesionales
- Uso de Grid y Flexbox
